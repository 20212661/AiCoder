# AiCoderX

A lightweight AI-powered code assistant that helps you navigate, understand, and modify codebases via natural language conversations.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/aiCoderX.git
   cd aiCoderX
   ```

2. Create and activate a virtual environment (recommended):
   ```bash
   python -m venv venv
   venv\Scripts\activate   # Windows
   source venv/bin/activate  # macOS/Linux
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the assistant:
```bash
python main.py
```

Then start typing natural‑language instructions to read, edit, or debug your project files.
